# javaAdvanced

## Summary
End project for the java class at EPITA(specialisation semester). 
Application is a webview of the previous created application with minimal modifcations and containing the same core functionality.

### Features
The application manages patient identities. 
 * Enables to Create, View, Edit and delete the patients. 
 * Using responsive design and AJAX to provide a good user experience.
 * Authorisation and different levels of rights.
 * Initial admin configuration on first startup.

### Used (web)technologies
Servlet&JSP&JSTL, AJAX, JSON, HTML&CSS&Bootstrap

### Java Frameworks
Frameworks used: Maven, Spring, Hibernate and JUnit.
